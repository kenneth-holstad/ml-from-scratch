ML from scratch repo by Kenneth Holstad

The purpose of this is to build functioning prototypes for machine learning problems using OOP principles in python.

Current Planned Scope: regression (linear and logistic), PCA, KNN, K-Means, MLP, CNN
Out of Scope: RNN's, ResNets/U-nets, Transformers

Currently functional: linear regression (SLR, MLR, OLS)
